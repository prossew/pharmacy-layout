# FMF - Complex SW Суставы

Лендинг продукта «Complex SW СУСТАВЫ» для компании Оптисалт.

🔗 Demo: https://prossew.github.io/pharmacy-layout/

## Технологии

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* БЭМ
* Pixel Perfect верстка
* Адаптивная верстка
* SVG graphics

---

# Реализовано

* Pixel Perfect верстка по макету
* Полностью адаптивный интерфейс
* Мобильное меню
* Слайдер карточек на мобильных устройствах
* Семантическая структура страницы
* Оптимизация изображений и ресурсов
* SEO-базовая оптимизация
